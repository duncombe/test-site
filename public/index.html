SOS Guidlines: WSDD & Tests

IOOS SOS Guidelines currently include IOOS SOS Profile 1.0 and IOOS SOS Test Set. The Profile determines the IOOS-compliant implementation of the OGC Standards, and Test Set allows to verify the compliance. The Profile includes a collection of the SOS operation templates, and is described in details in the IOOS SOS Web Service Description Document (WSDD). The IOOS SOS test collection includes both OGC Standard and IOOS Profile compliance tests.
